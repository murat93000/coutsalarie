2. Données collectées

CoûtSalarié Pro collecte uniquement les données strictement nécessaires à son fonctionnement :

  • Données saisies dans les formulaires (informations employeur, salarié, contrat) → stockées localement dans votre navigateur (localStorage)
  • Email de contact (optionnel) → utilisé uniquement pour l'envoi de votre document et les échanges support
  • Données de paiement → traitées exclusivement par Stripe (nous ne stockons aucune donnée bancaire)

3. Stockage des données

Toutes les données saisies dans l'application sont stockées localement dans votre navigateur via le mécanisme localStorage. Ces données :

  • Ne sont jamais transmises à nos serveurs
  • Restent sur votre appareil
  • Peuvent être supprimées à tout moment en vidant le cache de votre navigateur

4. Paiement sécurisé — Stripe

Les paiements sont traités par Stripe (stripe.com), plateforme certifiée PCI-DSS. Lors d'un achat :

  • Vous êtes redirigé vers la page de paiement sécurisée de Stripe
  • Nous ne voyons ni ne stockons vos informations bancaires
  • Stripe peut collecter votre email et les informations de paiement selon ses propres conditions

Politique de confidentialité de Stripe : stripe.com/fr/privacy

6. Cookies

CoûtSalarié Pro n'utilise pas de cookies publicitaires ou de tracking. Seul le mécanisme localStorage est utilisé pour sauvegarder vos préférences et données localement.

  • Stripe : pour le traitement des paiements
  • EmailJS : pour l'envoi d'emails de confirmation (email uniquement)
  • Obligation légale : si requis par la loi française
